## Публикация сертификатов

Идём к созданным ранее **корневым сертификатам** (Администрирование УКЦ). Создавались два новых сертификата — на обоих **ПКМ → Опубликовать**:

![Опубликовать корневые сертификаты](images/p59_135.jpeg)

Публикуем сертификаты пользователей:

![Публикация сертификатов пользователей](images/p60_136.jpeg)
![Публикация](images/p60_137.jpeg)

![Выбор пользователя](images/p61_138.jpeg)

Указываем пользователя (по нашему варианту — **NetAdmin**):

![NetAdmin](images/p61_139.jpeg)

Нажимаем **Проверить** → должно быть без ошибок:

![Проверка публикации](images/p62_140.jpeg)

Через некоторое время — информация об успешных публикациях:

![Успешные публикации](images/p62_141.jpeg)

## Регистрейшен (Registration Point)

Идём в **Регистрейшен**:

![Регистрейшен](images/p63_142.jpeg)

**Настройка → Поиск ключей сетевого узла**:

![Поиск ключей сетевого узла](images/p63_143.jpeg)
![Шаг](images/p63_144.jpeg)

Вводим знакомый пароль → вход.

![Вход](images/p64_145.jpeg)

**Сервис → Настройка** → ставим галочку и **Запускаем транспортный модуль**:

![Транспортный модуль](images/p64_146.jpeg)

Регистрируем пользователя:

![Регистрация пользователя](images/p64_147.jpeg)

![Пользователь зарегистрирован](images/p65_148.jpeg)

**ПКМ на пользователе → Сертификаты → Создать запрос**:

![Создать запрос](images/p65_149.jpeg)

Всё по умолчанию, известный пароль → запрос отправляется:

![Отправка запроса](images/p65_150.jpeg)

Можно закрыть окно, посмотреть **Сертификаты → Отправленные**. Идём в УКЦ → **Необработанные → Обновить** — там должен появиться запрос:

![Необработанные запросы](images/p66_151.jpeg)

> Если запрос не появился:
> - перезагрузить обе виртуальные машины;
> - если не помогло — создать новый запрос на сертификат.

Нажимаем **Удовлетворить**:

![Удовлетворить запрос](images/p66_152.jpeg)

Везде Далее → далее → далее.

![Шаг](images/p67_153.jpeg)

## Аннулирование сертификата

По заданию нужно **Аннулировать** выданный сертификат:

![Аннулирование сертификата](images/p67_154.jpeg)
![Аннулирование](images/p67_155.jpeg)

Далее в **УКЦ**:

![УКЦ — аннулирование](images/p68_156.jpeg)

Нажимаем **Удовлетворить** — информация появится в **Удовлетворённые запросы**:

![Удовлетворённые запросы](images/p68_157.jpeg)

> 📷 В отчёт нужны скриншоты: отправки запросов, удовлетворённые запросы на выдачу, удовлетворённые запросы на аннулирование сертификатов.
